# Second Hand Market

## Описание

SPA-приложение для просмотра товаров с фильтрацией по категориям и пагинацией, а также корзин товаров и возможностью общения в чате.

## Краткая функциональность

- Авторизация пользователей
- Просмотр каталога товаров
- Фильтрация каталога товаров
- Просмотр корзин с товарами в удобном формате
- Чат
- Адаптивный дизайн

## Список зависимостей

Основные зависимотсти:

- vue
- vue-router
- pinia
- vite
- primevue

## Структура проекта

В папке source(src) есть несколько папок, которые каждая отвечает за свою задачу:

- assets — Папка с картинками и иконками
- components — здесь хранятся кастомные компоненты, а также кастомные иконки в папке icons
- layouts — Тут хранятся компоненты, которые не могут быть переиспользованы. Т.е. они используются в одном месте
- router — папака с настройками vue-router
- stores — папка с настройками хранилищ
- services — тут прописаны все запросы к api
- styles — в этой папке хранятся стили
- views — тут хранятся компоненты которые использует роутер для отображения
- types — здесь пользовательские типы
- utils — вспомогательные функции приложения

## Инструкция по запуску

1. Клонируйте репозиторий

`git clone https://github.com/Lis1459/Second-Hand-Market.git`

2. Перейдите в папку с проектом

`cd Second-Hand-Market`

3. Запуск в режиме разработки

`npm run dev`

После запуска проект будет доступен на по адресу:

`http://localhost:5173`

4. Конечная сборка проекта (для продакшена)

`npm run build`
